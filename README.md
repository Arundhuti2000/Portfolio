# Personal Portfolio

Visit my portfolio at [arundhatidas.com](https://arundhatidas.com)

## Tech Stack
- React
- Vite
- TailwindCSS
- Firebase Hosting

## Features
- Responsive design
- Dark theme
- Interactive UI components
- Custom animations
- Contact form

## Local Development
```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment
Automated deployment via GitHub Actions to Firebase Hosting.
